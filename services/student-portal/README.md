# Student Portal

## Description
Interface web React pour les étudiants avec dashboard, modules, recommandations et ressources.

## Stack Technique
- **Framework**: React 18
- **Build Tool**: Vite
- **Routing**: React Router
- **Visualisation**: Chart.js

## Fonctionnalités

- **Dashboard**: Vue d'ensemble de la progression
- **Mes Modules**: Liste des modules inscrits
- **Recommandations**: Ressources recommandées personnalisées
- **Ressources**: Catalogue de ressources pédagogiques
- **Profil**: Informations et statistiques

## Installation

```bash
npm install
```

## Exécution

```bash
npm run dev
```

## Port

Le service s'exécute sur le port **3009**.
